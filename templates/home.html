Who I Am

I graduated from the University of South Carolina in 2015 with a degree in Exercise Science. In 2016, I enrolled in the backend engineering course at The Iron Yard to pursue a career in programming. Though rigorous, I grew to love coding even more throughout the course. I find the work I do challenging and rewarding. I am an avid fan of video games, board games, and music.

What I Do

I have created many applications at The Iron Yard from a simple operations calculator to a profile-based search engine. Though each project is of varying complexity, I bring each to life by utilizing the tools and skills I have learned.

Why I Do It

I enjoy solving problems. Just like in video games, you have objectives and goals to work for. Coding is no different. Every mistake and setback is a new opportunity for me to learn and grow as a developer. The satisfaction I get when I fix errors and bugs and create working code reminds me that programming is what I am passionate about.

Seekers

Seekers is a social platform for adopted people and their birth families to interact and potentially find each other. Users fill out a profile and can search for other profiles through keywords such as birth hospital and surnames. If a user has done genetic testing/analysis through 23 and Me, he or she can import relative information to be displayed on their profile for all users to see.

Featured Tools: Django Haystack, Elasticsearch, Python Social Auth, 23andMe API, Amazon S3, Semantic UI


Reddit-Ish

Reddit-Ish is an application created for an assignment based on Reddit. It was created with emphasis on class-based views and foreign key relationships. Bootstrap is used as the CSS framework.

Featured Tools: Python, Django, Bootstrap


URL Shortener

URL Shortener is an application created for an assignment based off of Bitly where one can create shortened URL links to share across the Internet. It features class-based views and Bootstrap as the CSS framework. Users to set their created links to public or private.

Featured Tools: Python, Django, Bootstrap
